# 问道玩家指南

本文面向玩家，说明当前版本已经开放的玩法、指令和常见流程。  
这里不展示战斗公式、掉落概率、修为收益公式等内部数值计算；需要调整数值时以游戏内实际显示为准。

## 一句话说明

《问道》是一款 Telegram 文字修仙 RPG。你会从测灵根开始，通过闭关、历练、秘境、炼丹炼器、装备功法、宗门和群内玩法逐步变强。

养成操作主要在机器人私聊中完成；群聊主要用于切磋、排行、世界 Boss 和宗门社交。

## 当前版本重点

当前版本是 v0.1.5，重点变化如下：

- 闭关可作为后台被动轨运行，历练、秘境和炼制可作为前台主动轨并行推进。
- 已稳定通关的本境界地图可扫荡，免去重复等待，但仍消耗精力并走原战斗 / 掉落口径。
- 历练可能触发奇遇选择；金丹 / 元婴大突破的天劫改为逐雷互动。
- 新增 `/quest` 悬赏与成就系统，用每日 / 周常目标给出上线方向。
- 群内会低频播报突破、斩妖王、稀有掉落、天梯段位变动和宗门升级等大事件。

## 快速开始

1. 在机器人私聊发送 `/start`。
2. 系统会为你创建角色，完成测灵根，并给出初始境界、根骨、灵石、初始法宝和初始战技。
3. 发送 `/me` 查看角色面板。
4. 发送 `/cultivate` 开始闭关，过一段时间后再次发送 `/cultivate` 出关收功。
5. 修为足够后，在 `/me` 或菜单中点击「尝试突破」。
6. 用 `/explore` 历练获取材料、灵石、丹药和突破所需物品。

建议新玩家先按这个顺序玩：

```text
/start → /me → /cultivate → /explore → /bag → /craft → /skills → /sect
```

## 私聊与群聊

### 私聊适合做什么

这些操作建议在机器人私聊中进行：

- 查看角色：`/me`
- 闭关：`/cultivate`
- 历练：`/explore`
- 秘境：`/dungeon`
- 炼丹炼器：`/craft`
- 储物袋：`/bag`
- 法宝与功法：`/skills`
- NPC 商店：`/shop`
- 每日签到：`/daily`
- 悬赏任务：`/quest`
- 帮助：`/help`

如果你在群里点了私聊限定操作，机器人会提示你回到私聊。

### 群聊适合做什么

这些玩法主要在群聊中进行：

- 切磋：`/pvp`
- 排行：`/rank`
- 世界 Boss：`/boss`

宗门相关指令也可以用于组织玩家，但创建、加入、任务和商店操作本身也可以通过指令完成。

## 角色面板 `/me`

`/me` 会显示你的当前状态，包括：

- 灵根与根骨
- 境界
- 修为进度
- 灵石
- 精力
- 法身六维
- 当前法宝
- 当前心法
- 当前战技
- 是否正在闭关
- 是否可以突破

如果修为已足，面板会出现「尝试突破」按钮。

## 闭关 `/cultivate`

闭关是获取修为的主要方式。

使用方式：

1. 发送 `/cultivate` 开始闭关。
2. 闭关期间不能历练、秘境或挑战 Boss；群内切磋不受闭关影响。
3. 再次发送 `/cultivate` 或点击闭关按钮，即可出关收功。

当前版本有离线结算上限。也就是说，单次闭关时间太长时，超出上限的部分不会继续增加收益。你可以分多次闭关来持续推进。

如果你一段时间没有操作，系统会在满足条件时自动帮你进入闭关状态。下一次你回来操作时，可以正常出关收功。

## 突破

当修为达到当前阶段要求后，可以尝试突破。

突破分两类：

- 小阶段突破：修为足够即可尝试。
- 大境界突破：需要对应突破丹。

当前境界路线：

```text
炼气期 → 筑基期 → 金丹期 → 元婴期
```

大境界突破需要的丹药：

- 进入筑基期：筑基丹
- 进入金丹期：金丹
- 进入元婴期：元婴丹

突破失败不会跌境，但可能损失一部分当前修为，并获得「道基不稳」状态。道基不稳会临时降低战斗表现，可以等待自然恢复，也可以使用疗伤丹处理。

## 精力

精力用于历练、秘境和部分群内战斗。

精力会随时间恢复。你也可以通过以下方式处理精力不足：

- 等待自然恢复。
- 在商店用灵石购买精力；按钮会显示下一次购买的价格、获得精力和今日上限。
- 加入宗门后获得更高精力上限。

注意：补灵丹现在恢复法力，不再恢复精力。

## 气血与法力

气血和法力是跨活动的资源，不再每次满血满蓝开打。

- 在 `/me`、`/explore`、`/dungeon` 都能看到当前 / 最大气血与法力。
- 两者都随时间自然恢复，等待与离线期间都会结算：法力恢复较快，气血较慢。
- 历练或秘境结束后剩余的气血 / 法力会保留到下一次活动；残血或缺蓝会影响下一场胜负，以及秘境能深入的层数。
- 活动开始时会记录出发气血 / 法力，本场战斗按出发状态结算；晚领取或中途想服丹都不会改变那一场的结果。
- 单场战斗达到回合上限时，若双方都未倒下，会按剩余气血百分比判胜负。
- 即使战败也只是「重伤而归」：气血不会低于上限的两成，修为和装备不会损失。
- 气血过低时，历练 / 秘境入口会提示休整（但不会强制阻止出战）。

想立刻恢复，可以服用丹药：

- 疗伤丹：恢复部分气血（约半数上限），并清除道基不稳。
- 补灵丹：恢复部分法力（约半数上限）。
- 大还丹：一次补满气血与法力，适合硬仗前补给。

疗伤丹、补灵丹在筑基前也有稳定来源：

- 炼气期：直接到 `/shop` 购买，筑基前半价（疗伤丹 25 灵石、补灵丹 30 灵石）。
- 筑基后：`/shop` 恢复全价（疗伤丹 50、补灵丹 60），但用 `/craft` 自炼通常更省，是更划算的主力来源。

如果你还有正在进行或已经完成但未领取的历练 / 秘境，需要先领取结算，再服用恢复丹药。

## 历练 `/explore`

历练用于获取灵石、材料、丹药和突破资源。

使用方式：

1. 发送 `/explore`。
2. 默认只显示当前大境界的三张地图（易 / 中 / 难）；点「📜 低阶历练」可展开低境界地图，用于刷低阶材料或补突破资源。
3. 选择一张地图开始历练，消耗对应精力。
4. 历练需要一段时间完成，时长与难度、遭遇场数挂钩（开始时即确定）。
5. 完成后系统会私聊提醒你。
6. 回到 `/explore` 领取结算。

历练期间不能同时开始另一场历练。若你点开 `/explore`，界面会显示当前历练是否完成。

每个大境界有三档地图，进入该境界后同时可选；难度越高，怪越强、精力越贵、失败概率越高，但灵石 / 修为 / 掉落更丰厚，困难图还有独占材料：

| 境界 | 易（稳定刷） | 中（约 1.5x 收益） | 难（约 2x 收益 + 独占掉落） |
| --- | --- | --- | --- |
| 炼气期 | 青牛后山 | 碧萝竹海 | 阴风古洞（阴风石 / 幽冥草） |
| 筑基期 | 妖兽森林 | 赤霞岩谷 | 枯骨沼泽（白骨精华 / 腐泽妖核） |
| 金丹期 | 万妖岭 | 碧毒蛟潭 | 九霄雷泽（雷纹玄铁 / 劫火残晶） |
| 元婴期 | 上古战场 | 归墟裂谷 | 天魔古原（天魔残页 / 古战魂晶） |

- 易：精力 10，约 8–10 分钟，1 场遭遇，稳定日常刷。
- 中：精力 12，约 10–13 分钟，1–2 场遭遇，妖王率更高。
- 难：精力 15，约 13–16 分钟，2–3 场遭遇，刚解锁多半打不动；随着修为提升会逐渐变成稳定主刷图。
- 失败不扣资源，但也不返还精力，所以难图属于高风险高回报。

## 秘境 `/dungeon`

秘境是更高强度的 PvE 挑战，会逐层战斗，并产出更稀有的材料、残页、图纸和法宝。

使用方式：

1. 发送 `/dungeon`。
2. 选择已解锁秘境。
3. 开始探索后消耗精力。
4. 秘境需要一段时间完成。
5. 完成后系统会私聊提醒你。
6. 回到 `/dungeon` 领取结算。

当前开放：

- 青云秘境：金丹后开放。

当前规则：

- 每个秘境每天最多进入 3 次。
- 同一时间只能有一个秘境探索进行中。
- 闭关中不能进入秘境。

秘境可能产出：

- 天材地宝
- 功法残页
- 丹方
- 图纸
- 法宝

## 储物袋 `/bag`

储物袋会显示你的材料、丹药、丹方、图纸和法宝。

可以直接使用的物品会显示「使用」按钮。当前可直接使用的物品包括：

- 疗伤丹：恢复部分气血，并清除道基不稳。
- 补灵丹：恢复部分法力。
- 大还丹：一次补满气血与法力。
- 虎力丹：获得临时战斗增益。
- 凝神丹：获得临时闭关增益。
- 洗髓丹：提升根骨。
- 天材地宝：提升根骨。
- 丹方和图纸：研读后解锁对应配方。

材料类物品通常不能直接使用。比如灵草主要用于炼丹，也可以在商店出售。

## 炼丹炼器 `/craft`

炼丹炼器用于把材料转化为丹药或法宝。

使用方式：

1. 发送 `/craft`。
2. 选择一个已解锁配方。
3. 系统会检查材料和灵石。
4. 开炉后需要等待一段时间，丹药通常是几分钟到十几分钟，炼器略久。
5. 再次打开 `/craft` 可领取出炉物品。
6. 如果炉中已有物品，可以使用灵石加速，费用按剩余分钟数计算。

当前默认可用配方包括：

- 疗伤丹
- 补灵丹
- 虎力丹
- 凝神丹
- 筑基丹
- 大还丹（金丹期可炼）
- 玄铁剑
- 青木甲

部分配方需要先获得并研读丹方或图纸：

- 洗髓丹丹方 → 解锁洗髓丹。
- 聚灵佩图纸 → 解锁聚灵佩。

丹方和图纸可以通过秘境、商店或宗门商店获得。

## 法宝与功法 `/skills`

`/skills` 用于管理装备和功法。

### 法宝

当前装备槽位：

- 主器 1 个
- 护体 1 个
- 饰品 2 个

法宝会影响你的战斗表现。部分法宝还会对突破、渡劫或战斗产生额外帮助。

炼器获得的法宝可能带有随机词条。词条会在战斗或属性中生效。

### 功法

功法分为心法和战技：

- 心法：被动生效，同时只能装备 1 个。
- 战技：进入战斗后使用，最多装备 3 个。

新角色默认拥有：

- 心法：吐纳诀
- 战技：快剑斩

功法残页集齐后，可以在 `/skills` 中学习。当前残页包括：

- 归元心法残页
- 烈火诀残页
- 回春术残页

## 商店 `/shop`

商店提供 NPC 购买和回收。

你可以：

- 购买基础材料、丹药、突破丹、丹方和图纸。其中疗伤丹、补灵丹炼气期即可购买，筑基前半价。
- 出售背包中的可回收物品。
- 用灵石购买精力。

部分商品会随着境界提升而开放。

## 每日签到 `/daily`

每天可以签到一次。签到会获得灵石，连续签到会记录连签天数。

元婴圆满且修为已满时，如果背包中没有化神丹，签到会额外获得 1 枚绑定化神丹，作为突破化神的保底补给。

## 宗门 `/sect`

宗门是玩家组织玩法。

常用指令：

```text
/sect
/sect create 宗门名
/sect join 宗门名
/sect task
/sect leave
/sect upgrade
/sect buy 物品名
```

创建宗门需要达到指定境界，并消耗灵石。加入宗门后可以：

- 完成宗门任务，获得个人贡献和灵石。
- 在宗门商店兑换残页、丹方、图纸等物品。
- 享受宗门等级带来的福利。
- 宗主可以消耗宗门贡献池升级宗门。

宗门商店当前提供：

- 归元心法残页
- 烈火诀残页
- 回春术残页
- 玄铁剑图纸
- 洗髓丹丹方
- 聚灵佩图纸

## 群内切磋 `/pvp`

切磋在群聊中进行。它只影响天梯积分和声望，不掉资源。

常用方式：

```text
/pvp
/pvp @道号
/pvp #排名
```

也可以回复某位玩家的消息发送 `/pvp`，向对方发起切磋。

流程：

1. 在群里发送 `/pvp`。
2. 系统会生成一条切磋确认。
3. 点击「确认切磋」后自动结算。

注意：

- 闭关不会阻止你发起或接受群内切磋。
- 境界差距过大时不能进入天梯切磋。
- 每天可切磋次数有限。

## 排行 `/rank`

`/rank` 会显示当前天梯排行，包括积分、胜负和声望。

## 世界 Boss `/boss`

世界 Boss 在群聊中进行。

使用方式：

1. 在群里发送 `/boss`。
2. 系统会显示当前 Boss 战况。
3. 点击「挑战 Boss」参与合击。
4. 点击「查看战况」刷新 Boss 状态和伤害榜。

Boss 玩法特点：

- 每个群独立。
- Boss 档位会参考本群已知修仙者的中位境界。
- Boss 血量会按本群已知修仙者人数缩放，小群不会吃满大群血量。
- Boss 战会记录伤害榜。
- Boss 被击败或到期后，会按参与情况发放奖励。
- 闭关中不能挑战 Boss。
- 挑战 Boss 需要精力。

系统也会定时在已记录的群中刷新 Boss，并在公告中提供挑战和查看战况按钮。

## 推荐日常循环

### 新手阶段

1. `/daily` 签到。
2. `/cultivate` 闭关。
3. 出关后 `/me` 查看是否可突破。
4. `/explore` 去青牛后山历练。
5. `/bag` 查看获得的材料。
6. `/craft` 炼制丹药或法宝。
7. `/skills` 装备新法宝或学习功法。

### 筑基后

1. 继续闭关和历练。
2. 通过妖兽森林获取更高阶资源。
3. 开始炼制和装备更好的法宝。
4. 创建或加入宗门。
5. 参与群内切磋和 Boss。

### 金丹后

1. 进入青云秘境。
2. 收集功法残页、丹方、图纸和高阶法宝。
3. 用宗门商店补充关键资源。
4. 准备更高境界突破所需丹药。

## 常见问题

### 灵草为什么不能直接使用？

灵草是材料，不是丹药。它主要用于炼丹，例如疗伤丹、补灵丹、虎力丹、凝神丹和筑基丹。

### 疗伤丹丹方在哪里？

当前版本不需要疗伤丹丹方。疗伤丹是默认配方，达到要求后可直接在 `/craft` 中炼制。

### 为什么我不能历练或进秘境？

常见原因：

- 正在闭关。
- 精力不足。
- 境界未达到地图或秘境要求。
- 已经有一场历练或秘境正在进行。
- 当天秘境次数已用完。

### 为什么不能服用疗伤丹、补灵丹或大还丹？

常见原因：

- 气血 / 法力已经满了。
- 你还有正在进行或已完成但未领取的历练 / 秘境。请先领取结算，再服丹恢复。

### 补灵丹为什么不恢复精力了？

补灵丹现在用于恢复法力。精力不足时，可以等待自然恢复、去商店购买精力，或通过宗门福利提高上限。

### 筑基前怎么回血回蓝？

炼气期还不能用 `/craft` 炼制疗伤丹、补灵丹（炼丹炼器在筑基后才开放）。在此之前，直接到 `/shop` 购买即可——这两味恢复丹筑基前半价（疗伤丹 25 灵石、补灵丹 30 灵石）。筑基后既能继续在商店购买，也能自炼，自炼通常更省。当然也可以等待气血 / 法力自然恢复。

### 为什么我不能切磋？

常见原因：

- 你在私聊中使用了 `/pvp`。
- 没有合适对手。
- 今日切磋次数已满。
- 与对手境界差距过大。

### 为什么出关没有继续获得更多收益？

单次闭关有离线结算上限。达到上限后继续放置，不会继续增加本次收益。可以出关后再次闭关。

### 为什么炼丹炼器没马上拿到东西？

炼丹炼器需要等待出炉，丹药通常是几分钟到十几分钟，炼器略久。再次打开 `/craft` 可以领取已完成的物品，也可以使用灵石加速。

### 为什么我没有收到历练或秘境完成提醒？

提醒通过机器人私聊发送。请确认你已经和机器人私聊 `/start` 过，并且没有屏蔽机器人。即使没有收到提醒，也可以手动打开 `/explore` 或 `/dungeon` 查看并领取。
