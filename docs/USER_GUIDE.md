# 问道玩家指南

本文面向玩家，说明当前版本已经开放的玩法、指令和常见流程。
这里不展示战斗公式、掉落概率、修为收益公式等内部数值计算；具体数值以游戏内实际显示为准。

## 一句话说明

《问道》是一款 Telegram 文字修仙 RPG。你会从测灵根开始，通过闭关、历练、秘境、炼丹炼器、装备功法、宗门、道途和群内玩法逐步变强。

养成操作主要在机器人私聊中完成；群聊主要用于切磋、排行、世界 Boss 和宗门社交。

## 当前版本重点

当前版本已开放到 v2 主线内容：

- 境界上限开放到化神圆满：炼气、筑基、金丹、元婴、化神五个大境界。
- 元婴起开放 `/path` 道途 / 转修，支持剑修、体修、丹修、器修、符阵五类方向。
- 化神圆满后开放 `/ascension` 飞升试炼与飞升被动，让封顶后的修为继续有去处。
- 历练、秘境、化神装备、化神丹、太虚天门、化神世界 Boss 已形成高境界 PvE 闭环。
- 新增 `/weekly` 周活动副本、活动商店、绑定材料和飞升点兑换。
- 新增 `/sectwar` 宗门战据点，宗门成员可争夺据点获取轻量加成和赛季道行。
- 新增 `/market` 玩家一口价坊市，支持非绑定可交易物品上架、购买和撤单。
- 资源系统已接入精力、气血、法力、道行、飞升点；战斗按出发血蓝快照结算。

## 快速开始

1. 在机器人私聊发送 `/start`。
2. 系统会为你创建角色，完成测灵根，并给出初始境界、根骨、灵石、初始法宝和初始战技。
3. 发送 `/me` 查看角色面板。
4. 发送 `/daily` 签到，拿一笔稳定灵石。
5. 发送 `/cultivate` 开始闭关；过一段时间后再次发送 `/cultivate` 出关收功。
6. 修为足够后，在 `/me` 或菜单中点击「尝试突破」。
7. 用 `/explore` 历练获取材料、灵石、丹药和突破所需物品。
8. 筑基后用 `/craft` 炼丹炼器，用 `/skills` 装备法宝和学习功法。
9. 加入宗门后，可以做宗门任务、买宗门商店物品、参与宗门战。

建议新玩家先按这个顺序玩：

```text
/start → /daily → /me → /cultivate → /explore → /bag → /craft → /skills → /sect
```

到元婴后，记得打开 `/path` 选择道途；到化神圆满后，再打开 `/ascension` 处理飞升点和被动。

## 私聊与群聊

### 私聊适合做什么

这些操作建议在机器人私聊中进行：

- 查看角色：`/me`
- 闭关：`/cultivate`
- 历练：`/explore`
- 秘境：`/dungeon`
- 炼丹炼器：`/craft`
- 储物袋：`/bag`
- 法宝与功法：`/skills`
- NPC 商店：`/shop`
- 每日签到：`/daily`
- 悬赏任务：`/quest`
- 道途 / 转修：`/path`
- 飞升：`/ascension`
- 周活动副本：`/weekly`
- 玩家坊市：`/market`
- 宗门战据点：`/sectwar`
- 帮助：`/help`

如果你在群里点了私聊限定操作，机器人会提示你回到私聊。

### 群聊适合做什么

这些玩法主要在群聊中进行：

- 切磋：`/pvp`
- 排行：`/rank`
- 世界 Boss：`/boss`

宗门 `/sect` 可以在私聊或群聊中使用；创建、加入、任务、捐输、商店和升级都通过指令或按钮完成。

## 角色面板 `/me`

`/me` 会显示你的当前状态，包括：

- 灵根与根骨
- 境界
- 修为进度
- 灵石
- 精力
- 当前气血 / 法力
- 法身六维：攻击、防御、身法、暴击等
- 当前法宝
- 当前心法
- 当前战技
- 是否正在闭关
- 已获得的部分成就
- 是否可以突破

如果修为已足，面板会出现「尝试突破」按钮。

## 闭关 `/cultivate`

闭关是获取修为的主要方式。

使用方式：

1. 发送 `/cultivate` 开始闭关。
2. 再次发送 `/cultivate` 或点击闭关按钮，即可出关收功。
3. 如果修为达到当前阶段上限，可以在 `/me` 中尝试突破。

当前闭关是后台被动轨。历练、秘境、炼制、周活动等前台行为会记录活动窗口，并折算闭关收益，避免后台和前台收益同时满额膨胀。世界 Boss 等部分玩法仍会要求你不在闭关中，具体以机器人提示为准。

当前版本有离线结算上限。单次闭关时间太长时，超出上限的部分不会继续增加本次收益。可以出关后再次闭关，分多次持续推进。

如果你一段时间没有操作，系统会在满足条件时自动帮你进入闭关状态。下一次你回来操作时，可以正常出关收功。

化神圆满后，修为会封顶；继续获得的溢出修为会按规则转为道行和飞升点。

## 突破

当修为达到当前阶段要求后，可以尝试突破。

突破分两类：

- 小阶段突破：修为足够即可尝试。
- 大境界突破：需要对应突破丹，并可能触发天劫 / 神魂劫。

当前境界路线：

```text
炼气期 → 筑基期 → 金丹期 → 元婴期 → 化神期
```

大境界突破需要的丹药：

| 目标境界 | 需要丹药 | 额外机制 |
| --- | --- | --- |
| 筑基期 | 筑基丹 | 无天劫 |
| 金丹期 | 金丹 | 天劫 |
| 元婴期 | 元婴丹 | 天劫 |
| 化神期 | 化神丹 | 神魂劫 |

金丹、元婴和化神大突破会进入多段互动。你可以选择护体法宝、护盾战技、服用大还丹、硬抗，或在化神神魂劫中选择凝神守一等应对方式。

突破失败不会跌境，但可能损失一部分当前修为，并获得「道基不稳」状态。道基不稳会临时降低战斗表现，可以等待自然恢复，也可以使用疗伤丹处理。

化神丹不会在 NPC 商店直接出售。常见来源包括天魔古原、天外古墟、化神丹残方、太虚天门和化神丹方炼制链路。元婴圆满且修为已满时，如果背包里没有化神丹，每日签到会额外补给 1 枚绑定化神丹。

## 精力

精力用于历练、秘境、周活动、世界 Boss 和部分群内战斗。

精力会随时间恢复。你也可以通过以下方式处理精力不足：

- 等待自然恢复。
- 在 `/shop` 用灵石购买精力；每日购买次数有限，价格会递增。
- 加入宗门后获得更高精力上限。

注意：补灵丹恢复法力，不恢复精力。

## 气血与法力

气血和法力是跨活动的资源，不再每次满血满蓝开打。

- 在 `/me`、`/explore`、`/dungeon` 都能看到当前 / 最大气血与法力。
- 两者都随时间自然恢复，等待与离线期间都会结算：法力恢复较快，气血较慢。
- 历练或秘境结束后，剩余气血 / 法力会保留到下一次活动；残血或缺蓝会影响下一场胜负，以及秘境能深入的层数。
- 活动开始时会记录出发气血 / 法力，本场战斗按出发状态结算；晚领取或中途想服丹都不会改变那一场的结果。
- 单场战斗达到回合上限时，若双方都未倒下，会按剩余气血百分比判胜负。
- 即使战败也只是「重伤而归」：气血不会低于上限的两成，修为和装备不会损失。
- 气血过低时，历练 / 秘境入口会提示休整，但不会强制阻止出战。

想立刻恢复，可以服用丹药：

- 疗伤丹：恢复部分气血，并清除道基不稳。
- 补灵丹：恢复部分法力。
- 大还丹：一次补满气血与法力，适合硬仗前补给。

疗伤丹、补灵丹在筑基前也有稳定来源：

- 炼气期：直接到 `/shop` 购买，筑基前半价。
- 筑基后：`/shop` 恢复全价，但用 `/craft` 自炼通常更省。

如果你还有正在进行或已经完成但未领取的历练 / 秘境，需要先领取结算，再服用恢复丹药。

## 历练 `/explore`

历练用于获取灵石、材料、丹药和突破资源。

使用方式：

1. 发送 `/explore`。
2. 默认只显示当前大境界的三张地图（易 / 中 / 难）。
3. 点「低阶历练」可展开低境界地图，用于刷低阶材料或补突破资源。
4. 选择一张地图开始历练，消耗对应精力。
5. 历练需要一段时间完成，时长与难度、遭遇场数挂钩。
6. 完成后系统会私聊提醒你。
7. 回到 `/explore` 领取结算。

每个大境界有三档地图。进入该境界后，本境界三张地图同时可选；难度越高，怪越强、精力越贵、失败概率越高，但灵石 / 修为 / 掉落更丰厚，困难图还有独占掉落。

| 境界 | 易（稳定刷） | 中（收益更高） | 难（高风险 + 独占掉落） |
| --- | --- | --- | --- |
| 炼气期 | 青牛后山 | 碧萝竹海 | 阴风古洞（阴风石 / 幽冥草） |
| 筑基期 | 妖兽森林 | 赤霞岩谷 | 枯骨沼泽（白骨精华 / 腐泽妖核） |
| 金丹期 | 万妖岭 | 碧毒蛟潭 | 九霄雷泽（雷纹玄铁 / 劫火残晶） |
| 元婴期 | 上古战场 | 归墟裂谷 | 天魔古原（天魔残页 / 古战魂晶 / 化神丹） |
| 化神期 | 星陨海 | 幽都裂隙 | 天外古墟（天外残玉 / 化神丹） |

难度大致规则：

- 易：约 8-10 分钟，1 场遭遇，稳定日常刷。
- 中：约 10-13 分钟，1-2 场遭遇，妖王率更高。
- 难：约 13-16 分钟，2-3 场遭遇，刚解锁多半打不动；随着修为提升会逐渐变成稳定主刷图。
- 失败不扣修为和装备，但不返还精力。

### 扫荡

本境界地图连续胜利 3 次后会解锁「扫荡」。扫荡会立即结算，但仍消耗精力、仍复用战斗和掉落规则，也会记录前台活动窗口用于折算闭关收益。

扫荡只针对当前大境界地图；低阶地图主要用于刷材料和补突破资源。

### 奇遇

历练中可能触发低频奇遇，例如坠崖洞府。奇遇会给出多个选择：

- 冒险探入：收益更高，但可能先损气血。
- 稳妥绕行：按常规历练结算。
- 救人：收益较低，但可获得宗门贡献等额外好处。

## 秘境 `/dungeon`

秘境是更高强度的 PvE 挑战，会逐层战斗，并产出更稀有的材料、残页、丹方、图纸和法宝。

使用方式：

1. 发送 `/dungeon`。
2. 选择已解锁秘境。
3. 开始探索后消耗精力，部分秘境还会收取入场费。
4. 秘境固定约 30 分钟完成。
5. 完成后系统会私聊提醒你。
6. 回到 `/dungeon` 领取结算。

当前开放秘境：

| 秘境 | 解锁境界 | 层数 | 精力 | 每日次数 | 入场费 |
| --- | --- | ---: | ---: | ---: | ---: |
| 灵溪洞 | 炼气期 | 3 | 33 | 3 | 0 |
| 玄冥矿窟 | 筑基期 | 4 | 40 | 3 | 80 |
| 青云秘境 | 金丹期 | 5 | 50 | 3 | 300 |
| 天墟古阵 | 元婴期 | 5 | 66 | 3 | 800 |
| 太虚天门 | 化神期 | 5 | 60 | 2 | 1800 |

当前规则：

- 同一时间只能有一个秘境探索进行中。
- 正在历练时不能进入秘境。
- 每个秘境有独立每日次数限制。
- 秘境逐层结算，能打到第几层就拿对应比例收益。
- 太虚天门是化神期重要来源，可产出化神丹残方、转修令、化神装备图纸和化神法宝。

## 储物袋 `/bag`

储物袋会显示你的材料、丹药、丹方、图纸和法宝。

可以直接使用的物品会显示「使用」按钮。当前可直接使用的物品包括：

- 疗伤丹：恢复部分气血，并清除道基不稳。
- 补灵丹：恢复部分法力。
- 大还丹：一次补满气血与法力。
- 虎力丹：获得临时战斗增益。
- 凝神丹：获得临时闭关增益。
- 保命符：获得临时生存增益。
- 洗髓丹：提升根骨。
- 天材地宝：提升根骨。
- 丹方和图纸：研读后解锁对应配方。

材料类物品通常不能直接使用。比如灵草主要用于炼丹，也可以在商店出售。

部分物品是绑定物，绑定物不能在玩家坊市交易。

## 炼丹炼器 `/craft`

炼丹炼器用于把材料转化为丹药或法宝。

使用方式：

1. 发送 `/craft`。
2. 选择一个已解锁配方。
3. 系统会检查材料和灵石。
4. 开炉后需要等待一段时间，丹药通常是几分钟到十几分钟，高阶炼器更久。
5. 再次打开 `/craft` 可领取出炉物品。
6. 如果炉中已有物品，可以使用灵石加速，费用按剩余分钟数计算。

当前默认可用配方包括：

- 疗伤丹
- 补灵丹
- 虎力丹
- 凝神丹
- 筑基丹
- 大还丹
- 玄铁剑
- 青木甲

部分配方需要先获得并研读丹方或图纸：

- 洗髓丹丹方 → 解锁洗髓丹。
- 聚灵佩图纸 → 解锁聚灵佩。
- 化神丹方 → 解锁化神丹。
- 陨星剑图纸、幽都甲图纸、太虚佩图纸 → 解锁化神法宝。

丹方和图纸可以通过秘境、商店、宗门商店或高阶内容获得。

炼器获得的法宝可能带有随机词条。器修道途会提高炼器相关收益。

## 法宝与功法 `/skills`

`/skills` 用于管理装备、功法和法宝长线养成。

### 法宝

当前装备槽位：

- 主器 1 个
- 护体 1 个
- 饰品 2 个

法宝会影响你的战斗表现。部分法宝还会对突破、渡劫或战斗产生额外帮助。

法宝实例可进行：

- 装备：放入对应槽位。
- 强化：消耗灵石和器魂，提升法宝基础属性。
- 重铸：消耗灵石和器魂，重 roll 法宝词条。
- 分解：未装备法宝可分解为器魂。

### 功法

功法分为心法和战技：

- 心法：被动生效，同时只能装备 1 个。
- 战技：进入战斗后使用，最多装备 3 个。

新角色默认拥有：

- 心法：吐纳诀
- 战技：快剑斩

功法残页集齐后，可以在 `/skills` 中学习。当前残页包括：

- 归元心法残页
- 烈火诀残页
- 回春术残页

战技需要消耗法力。法力不足时，战斗中可能多次转为普通攻击，输出会下降。

## 商店 `/shop`

商店提供 NPC 购买、回收和灵石买精力。

你可以：

- 购买基础材料、丹药、突破丹、丹方和图纸。
- 出售背包中的可回收物品。
- 用灵石购买精力。

部分商品会随着境界提升而开放。疗伤丹、补灵丹在炼气期半价，筑基后恢复全价。

NPC 商店不会直接出售化神丹、转修令、保命符等关键高阶物品。

## 每日签到 `/daily`

每天可以签到一次。签到会获得灵石，连续签到会记录连签天数。

元婴圆满且修为已满时，如果背包中没有化神丹，签到会额外获得 1 枚绑定化神丹，作为突破化神的保底补给。

## 悬赏任务 `/quest`

悬赏任务给你每天和每周的上线目标。

当前包括：

- 日常：历练胜利、完成炼制、天梯胜利等。
- 周常：秘境通关、挑战世界 Boss 等。
- 成就：金丹初成、元婴出窍、初斩妖王、天梯首胜、秘境通关等。

完成后回到 `/quest` 领取奖励。部分成就也会展示在 `/me`。

## 宗门 `/sect`

宗门是玩家组织玩法。

常用指令：

```text
/sect
/sect create 宗门名
/sect join 宗门名
/sect task
/sect donate 数量
/sect leave
/sect upgrade
/sect buy 物品名
```

创建宗门需要筑基后，并消耗灵石。加入宗门后可以：

- 完成宗门任务，获得个人贡献和灵石。
- 捐输灵石换贡献，贡献每日有上限。
- 在宗门商店兑换残页、丹方、图纸等物品。
- 享受宗门等级带来的福利，例如精力上限、闭关效率、掉落加成和离线上限。
- 宗主可以消耗宗门贡献池和灵石升级宗门。

宗门商店当前提供：

- 归元心法残页
- 烈火诀残页
- 回春术残页
- 玄铁剑图纸
- 洗髓丹丹方
- 聚灵佩图纸

## 道途 `/path`

道途是元婴起开放的专修长线。达到元婴初期后，可以在 `/path` 中选择第一条道途，首条免费。

当前五类道途：

| 道途 | 定位 |
| --- | --- |
| 剑修 | 攻伐与 Boss 伤害 |
| 体修 | 生存与秘境推进 |
| 丹修 | 丹药与突破辅助 |
| 器修 | 法宝与强化 |
| 符阵 | 闭关与洞府效率 |

道途等级：

```text
入门 → 小成 → 大成 → 圆满 → 宗师
```

升阶会消耗道行，后续阶段还会消耗化神材料：

- 小成：道行。
- 大成：道行 + 星陨砂。
- 圆满：道行 + 幽都魂晶。
- 宗师：道行 + 天外残玉。

第一次选择道途免费。之后切换主道途需要转修：

- 需要绑定转修令。
- 需要消耗灵石。
- 有转修冷却。
- 已经修过的道途进度会保留。

道行常见来源包括周活动副本、元婴 / 化神封顶后的溢出修为转换、PvP 月赛季奖励和宗门战赛季奖励。

## 飞升 `/ascension`

飞升是化神圆满后的长期成长。

打开 `/ascension` 可以看到：

- 当前飞升点。
- 飞升总阶。
- 已解锁尊号。
- 当前道行。
- 可挑战的飞升试炼。
- 可升级的飞升被动。

飞升试炼规则：

- 需要化神圆满。
- 每周只能完成一次。
- 消耗道行，获得飞升点。

飞升被动：

- 气血
- 攻伐
- 护体
- 闭关

每个被动都有等级上限，每级提供小幅加成。飞升总阶提升后会解锁对应尊号。

飞升点常见来源：

- 化神圆满后的溢出修为转换。
- 每周飞升试炼。
- 化神世界 Boss 前列奖励。
- 周活动商店兑换。

## 周活动 `/weekly`

周活动副本是周期性道行和绑定材料来源。

规则：

- 每周开放一个主题，例如天魔潮、丹霞会、万剑冢。
- 每次挑战消耗精力。
- 完成后获得道行和本周主题绑定材料。
- 道行有每周上限。
- 活动材料可在活动商店兑换绑定保命符或飞升点。

周活动是元婴以后推进道途、化神以后推进飞升的重要日常内容之一。

## 宗门战 `/sectwar`

宗门战据点通过 `/sectwar` 进入，当前为私聊入口。

规则：

- 需要先加入宗门。
- 开放时间为每周六 20:00-21:00（上海时间）。
- 玩家选择据点后，会先挑战据点守卫。
- 击败守卫后，为宗门累计据点积分。
- 月末赛季结算时，积分最高宗门成员获得道行奖励。

当前据点：

| 据点 | 加成方向 |
| --- | --- |
| 洞府据点 | 闭关效率 |
| 矿脉据点 | 掉落加成 |
| 祭坛据点 | 法身属性 |

据点加成为轻量加成，会进入全局合算上限。

## 玩家坊市 `/market`

坊市是玩家一口价交易入口，当前在私聊中使用。

你可以：

- 查看当前在售挂单。
- 购买其他玩家挂单。
- 上架自己的非绑定可交易物品。
- 撤回自己的挂单。

当前规则：

- 上架会从储物袋中扣除物品，撤单会返还。
- 购买会支付灵石，卖家获得扣税后的灵石。
- 当前主要交易背包里的材料 / 丹药等堆叠物品，不交易法宝实例。
- 绑定物不可上架。
- 化神丹、转修令、保命符等关键物品即使以非绑定形式进入背包，也不可上架。

## 群内切磋 `/pvp`

切磋在群聊中进行。它只影响天梯积分和声望，不掉资源。

常用方式：

```text
/pvp
/pvp @道号
/pvp #排名
```

也可以回复某位玩家的消息发送 `/pvp`，向对方发起切磋。

流程：

1. 在群里发送 `/pvp`。
2. 系统会生成一条切磋确认。
3. 点击「确认切磋」后自动结算。

注意：

- 闭关不会阻止你发起或接受群内切磋。
- 境界差距过大时不能进入天梯切磋。
- 每天可切磋次数有限。
- 同一对手每日只计一次有效声望，降低互刷价值。
- 天梯积分即时变化；灵石奖励改为按周声望排名结算奖池。

## 排行 `/rank`

`/rank` 会显示当前天梯排行，包括积分、段位、声望、胜负记录。

## 世界 Boss `/boss`

世界 Boss 在群聊中进行。

使用方式：

1. 在群里发送 `/boss`。
2. 系统会显示当前 Boss 战况。
3. 点击「挑战 Boss」参与合击。
4. 点击「查看战况」刷新 Boss 状态和伤害榜。

Boss 玩法特点：

- 每个群独立。
- Boss 档位会参考本群已知修仙者的中位境界。
- Boss 血量会按本群已知修仙者人数缩放，小群不会吃满大群血量。
- Boss 战会记录伤害榜。
- Boss 被击败或到期后，会按参与情况发放奖励。
- 闭关中不能挑战 Boss。
- 挑战 Boss 需要精力。
- 化神档 Boss 前列奖励可获得飞升点。

系统也会定时在已记录的群中刷新 Boss，并在公告中提供挑战和查看战况按钮。

## 物品说明

### 材料

- 灵草：基础炼丹材料。
- 玄铁矿：炼器材料。
- 妖兽皮：炼器材料。
- 妖丹：炼丹和突破相关材料。
- 天材地宝：可直接炼化提升根骨，也可用于高级炼丹炼器。
- 阴风石、幽冥草、白骨精华、腐泽妖核、雷纹玄铁、劫火残晶：各境界困难图独占材料。
- 天魔残页、古战魂晶：元婴困难图材料。
- 星陨砂、幽都魂晶、天外残玉：化神材料，可用于道途升阶、高阶炼制等。
- 化神丹残方：化神丹炼制链路材料。
- 转修令：用于切换主道途，默认作为关键物品隔离交易。
- 活动材料：天魔令、丹霞玉、剑冢铁，可在周活动商店兑换。

### 丹药

- 疗伤丹：恢复部分气血，兼清道基不稳。
- 补灵丹：恢复部分法力。
- 大还丹：一次补满气血与法力。
- 虎力丹：获得临时战斗增益。
- 凝神丹：获得临时闭关增益。
- 保命符：获得临时生存增益。
- 洗髓丹：提升根骨。
- 筑基丹、金丹、元婴丹、化神丹：用于大境界突破。

### 法宝

- 新手木剑：初始主器。
- 玄铁剑：筑基档主器。
- 青木甲：筑基档护体。
- 聚灵佩：金丹档饰品。
- 陨星剑：化神档主器。
- 幽都甲：化神档护体。
- 太虚佩：化神档饰品。

### 残页、丹方、图纸

- 残页用于学习功法。
- 丹方用于解锁炼丹配方。
- 图纸用于解锁炼器配方。

## 推荐日常循环

### 新手阶段

1. `/daily` 签到。
2. `/cultivate` 闭关。
3. 出关后 `/me` 查看是否可突破。
4. `/explore` 去青牛后山、碧萝竹海或阴风古洞历练。
5. `/bag` 查看获得的材料。
6. 筑基后用 `/craft` 炼制丹药或法宝。
7. `/skills` 装备新法宝或学习功法。

### 筑基后

1. 继续闭关和历练。
2. 进入玄冥矿窟、妖兽森林等内容获取更高阶资源。
3. 开始炼制和装备更好的法宝。
4. 创建或加入宗门。
5. 参与群内切磋和 Boss。

### 金丹后

1. 进入青云秘境。
2. 收集功法残页、丹方、图纸和高阶法宝。
3. 用宗门商店补充关键资源。
4. 准备元婴突破所需丹药。

### 元婴后

1. 选择 `/path` 道途。
2. 刷元婴三图和天墟古阵，积累元婴材料。
3. 收集化神丹、化神丹残方和相关配方。
4. 参加周活动，积累道行。
5. 准备元婴圆满到化神的神魂劫。

### 化神后

1. 刷星陨海、幽都裂隙、天外古墟。
2. 挑战太虚天门，收集化神装备和高阶材料。
3. 参与化神世界 Boss，争取前列奖励。
4. 升阶道途，准备飞升长期成长。

### 化神圆满后

1. 继续闭关，把溢出修为转为道行和飞升点。
2. 每周完成飞升试炼。
3. 用周活动材料兑换飞升点。
4. 升级飞升被动。
5. 参与宗门战和天梯赛季，获取周期性道行。

## 常见问题

### 灵草为什么不能直接使用？

灵草是材料，不是丹药。它主要用于炼丹，例如疗伤丹、补灵丹、虎力丹、凝神丹和筑基丹。

### 疗伤丹丹方在哪里？

当前版本不需要疗伤丹丹方。疗伤丹是默认配方，达到要求后可直接在 `/craft` 中炼制。

### 为什么我不能历练或进秘境？

常见原因：

- 精力不足。
- 境界未达到地图或秘境要求。
- 已经有一场历练或秘境正在进行。
- 当天秘境次数已用完。
- 秘境入场费不足。

### 为什么不能服用疗伤丹、补灵丹或大还丹？

常见原因：

- 气血 / 法力已经满了。
- 你还有正在进行或已完成但未领取的历练 / 秘境。请先领取结算，再服丹恢复。

### 补灵丹为什么不恢复精力了？

补灵丹现在用于恢复法力。精力不足时，可以等待自然恢复、去商店购买精力，或通过宗门福利提高上限。

### 筑基前怎么回血回蓝？

炼气期还不能用 `/craft` 炼制疗伤丹、补灵丹。在此之前，直接到 `/shop` 购买即可；这两味恢复丹筑基前半价。筑基后既能继续在商店购买，也能自炼，自炼通常更省。当然也可以等待气血 / 法力自然恢复。

### 为什么我不能切磋？

常见原因：

- 你在私聊中使用了 `/pvp`。
- 没有合适对手。
- 今日切磋次数已满。
- 与对手境界差距过大。

### 为什么出关没有继续获得更多收益？

单次闭关有离线结算上限。达到上限后继续放置，不会继续增加本次收益。可以出关后再次闭关。

### 化神圆满后继续闭关有什么用？

化神圆满后修为会封顶，但溢出的修为会按规则转为道行和飞升点，用于道途、飞升试炼和飞升被动。

### 为什么炼丹炼器没马上拿到东西？

炼丹炼器需要等待出炉，丹药通常是几分钟到十几分钟，高阶炼器更久。再次打开 `/craft` 可以领取已完成的物品，也可以使用灵石加速。

### 为什么我没有收到历练或秘境完成提醒？

提醒通过机器人私聊发送。请确认你已经和机器人私聊 `/start` 过，并且没有屏蔽机器人。即使没有收到提醒，也可以手动打开 `/explore` 或 `/dungeon` 查看并领取。

### 为什么我不能上架某个物品？

常见原因：

- 物品是绑定物。
- 物品属于限售关键物品，例如化神丹、转修令、保命符。
- 物品是法宝实例，当前坊市主要交易背包里的堆叠物品。
- 你没有足够数量的非绑定物品。

### 为什么我不能转修？

常见原因：

- 境界还未达到元婴。
- 缺少绑定转修令。
- 灵石不足。
- 转修冷却未结束。
- 当前道途已经是目标道途。

### 为什么我不能挑战飞升试炼？

常见原因：

- 还未达到化神圆满。
- 本周已经完成过飞升试炼。
- 道行不足。

### 为什么我不能参加宗门战？

常见原因：

- 尚未加入宗门。
- 当前不是周六 20:00-21:00。
- 据点守卫太强，战斗失败。
