# 《问道》二期设计规格 (spec-v2)

> 在当前 main / v0.1.5 玩法基础上的二期设计。
> 二期的第一目标不是“横向加系统”，而是解决已经出现的满级空窗：
> **开放化神期 + 建立元婴期开启的道途/转修长线**，让挂机产出持续有去处。

文档状态:**设计草案，待评审后实施**。本文件用于替代旧 `v0.2` 分支里的 `spec-v2.md`。
数值标注“初版可调”者均需配合 `tools/balance_sim.py` 与上线数据继续调参。

---

## 0. 二期已锁定的方向

| # | 决定 | 取值 |
|---|---|---|
| 1 | 首要问题 | **已有玩家达到元婴上限，必须先提供继续成长目标** |
| 2 | 境界上限 | v2 从元婴圆满提升到 **化神圆满**；炼虚及以上留三期 |
| 3 | 专修长线 | 新增 **道途 / 转修 / 飞升点**；元婴期开始可专修，封顶后继续沉淀 |
| 4 | 玩家存档 | **不重置、不回档**；现有元婴圆满玩家可直接接入新内容 |
| 5 | 经济优先级 | 玩家坊市仍做，但从二期第一优先级下调；先救满级体验 |
| 6 | 变现 | 暂不引入付费；尤其不在转修、突破、交易上做 pay-to-win |
| 7 | PvP 后果 | 普通切磋仍无资源惩罚；高风险 PvP 延后，不阻塞 v2.0 |
| 8 | 群玩法 | 维持同群独立 Boss / 宗门生态，不做跨群大服战 |

设计口径：

- **短期止血**：先开放化神期、化神地图、化神突破材料来源，使当前封顶玩家马上有修炼目标。
- **中期留存**：道途系统让元婴期及以上玩家开始“专修/转职”，同境界也能形成 build 差异。
- **长期挂机价值**：化神圆满后，多余修为转换为道行/飞升点，不再出现“挂机无意义”。

## 1. 接驳当前 main 的真实实现

当前 main 已经不只是 v1 定稿状态，二期必须复用这些后续改动：

| 现有能力 | 二期复用方式 |
|---|---|
| `config.realms` 的境界配置 | 追加化神期、化神突破丹、化神阶段时长、属性锚点 |
| 每境界三档历练图 | 给化神补 易/中/难 三张图，沿用难度、时长、遭遇数模型 |
| 前台活动窗口 `activity_windows` | 新增采集、活动副本、宗门战行动时都记录窗口，折算闭关收益 |
| 气血/法力资源化 | 化神内容按出发血蓝快照结算，避免满血假象 |
| 交互式天劫 `tribulation_sessions` | 元婴→化神改为“神魂劫”三段选择，复用状态机 |
| 世界 Boss 分档与人数缩放 | 增加化神 Boss 档，并继续按本群已知修仙者人数缩放 |
| 周 PvP 奖池 | 可平滑升级为赛季制，但不阻塞化神上线 |
| `callback_tokens` 与 `db.transaction()` | 后续坊市/拍卖/转修确认都必须走一次性 token + 原子事务 |
| `game_events` 领域事件 | 化神突破、道途升阶、飞升成功进入任务/成就/社交播报 |

当前实现中需要特别注意的约束：

- `reserve_stamina_for_action()` 会拒绝闭关中行动；二期新前台动作若允许后台闭关并行，应仿照历练/秘境记录 `activity_windows`，不要直接复用该函数。
- `inventory` 当前主键是 `(user_id, item_key)`；若要做绑定/非绑定共存，不能简单给该表加单个 `bound` 列。
- 经济反套利空间较窄，任何可交易产出、采集产出、洞府加成都要进新的平衡测试。

## 2. 二期核心循环

```
  元婴期
      ↓
  [道途] 剑修、体修、丹修、器修、符阵 五类专精
      ↓                         ↘
  [继续突破] 元婴圆满 → 化神突破   [转修] 保留已有道途积累，切换主道途形成不同 build
      ↓
  [化神期] 新地图 / 新秘境 / 化神 Boss / 高阶炼制
      ↓
  [化神圆满后] 多余修为 → 道行 / 飞升点
      ↓
  [飞升试炼] 消耗长期资源，换账号级小幅加成与称号
      ↺
  继续挂机、历练、Boss、宗门战、活动副本
```

这条线的设计目标是：即使玩家暂时到达版本最高境界，日常闭关、历练、Boss、炼制仍能推进道途或飞升点。

## 3. 境界上限扩展：化神期

### 3.1 新境界结构

| 大境界 | 小阶 | 主要解锁 |
|---|---|---|
| 元婴期 | 初 / 中 / 后 / 圆满 | 当前 v1 上限 |
| **化神期** | 初 / 中 / 后 / 圆满 | 道途进阶、神魂地图、化神秘境、化神 Boss、飞升试炼前置 |
| 炼虚期+ | 待定 | 三期 |

配置改动：

- `REALM_NAMES` 追加 `化神期`。
- `REALM_STAGES[4] = ["初期", "中期", "后期", "圆满"]`。
- `STAMINA_CAP[4] = 240` 初版。
- `STAMINA_BUY_BASE[4] = 2800` 初版，即当日首买 20 精力约 140 灵石/精力。
- `SECLUSION_STAGE_HOURS[4] = 96` 初版，让化神每小阶约 4 天基础闭关目标时长。
- `_REALM_BASE_COST[4] = 450000`，`_STAGE_MULT` 可保留 1.20 或化神单独改 1.25。

### 3.2 化神属性锚点

当前元婴圆满基础锚点是 `hp=16000 / mp=1500 / atk=1300 / df=950 / spd=380 / crit=140`。
化神不应只做数值膨胀，但需要让新内容有清晰门槛。

| 境界 | 气血 | 法力 | 攻击 | 防御 | 身法 | 暴击 |
|---|---:|---:|---:|---:|---:|---:|
| 化神初期 | 24000 | 2200 | 1850 | 1350 | 500 | 175 |
| 化神圆满 | 52000 | 4200 | 3800 | 2800 | 900 | 280 |

平衡目标：

- 当前战斗引擎近二值，胜率通常是 0% 或 100%，过渡带很窄；化神内容不以“偶尔输/偶尔赢”描述，而以**可刷 / 门槛**描述。
- 元婴圆满含当前满配、常用丹药、宗门福利时，可刷化神易图普通小怪；化神中/难图的 Boss 与多遭遇连战应作为离散门槛，压住元婴圆满满 buff 档。
- 化神初期能刷易图；中图、难图分别作为化神中期、化神后期的成长门槛。
- 化神圆满打旧元婴内容应明显碾压，但旧内容收益不应成为最佳成长路线。

装备验收口径：

- M0 没有化神装备，化神三张图的验收 profile 是 **化神初期锚点 + 现役元婴装备/功法**，不能因为缺化神装备而阻塞 M0。
- 当前 `tools.balance_sim.GEARED` 仍是以金丹/元婴阶段装备为主的“满配无词条”档；M1 化神装备上线后，再新增化神 GEARED 档并回归重调新内容。
- 同时新增 `YUANYING_FULL_BUFF` 或等价档位，用来模拟元婴圆满 + 当前可叠满 buff + **单条道途拉满** 的属性，所有化神中/难门槛都按该档调参。
- 满 buff 档至少拆出攻击极端与生存极端：如剑修圆满档验攻击/破防门槛，体修圆满档验连战/承伤门槛；这些道途加成都必须进入 §6.3 的合算倍率上限。
- 允许“元婴圆满满 buff 能磨化神易图”，不允许“元婴圆满满 buff 稳定刷化神中/难 Boss”。

### 3.3 元婴→化神突破

新增大突破：

| 目标境界 | 丹药 | 基础成功率 | 额外试炼 |
|---|---|---:|---|
| 化神期 | 化神丹 | 50% | 神魂劫 |

规则：

- 触发条件：元婴圆满 + 当前修为达到突破成本 + 化神丹。
- 成功率公式沿用现有 `big_success_rate` 结构：基础率 + 根骨修正 + 法宝/道途/丹药加成，上限 95%。
- 默认失败仍是轻惩罚：损失当前修为 30%，获得道基不稳，不跌境。
- 神魂劫复用 `tribulation_sessions`，仍为 3 段交互，但文案和选项改为神魂主题。

神魂劫选项初版：

| 选项 | 效果 |
|---|---|
| 凝神守一 | 稳定减伤，消耗法力 |
| 祭出护体法宝 | 高减伤，受装备加成影响 |
| 服大还丹 | 回复气血法力，消耗大还丹 |
| 放开心魔 | 高风险，若撑过本段则额外提高最终奖励 |

### 3.4 化神丹来源

化神丹必须在 v2.0 同步提供来源，否则只是纸面开放上限。

来源初版：

- 元婴困难图 `天魔古原` 极低概率掉落 `化神丹残方` 或 `化神丹`。
- 新化神易图低概率掉落化神丹材料。
- 新秘境 `太虚天门` 掉落完整丹方/材料。
- 世界 Boss 化神档参与奖不直接掉完整丹，但前列奖励可掉材料。
- NPC 商店暂不直接卖化神丹，避免用灵石跳过版本核心目标。

## 4. 道途 / 转修：元婴期开启的专修长线

### 4.1 解锁与定位

道途是二期的“转职”系统，解决两个问题：

- 元婴期玩家可以在继续突破境界的同时选择一条专修方向，形成 build 多样性。
- 元婴圆满玩家即使还未突破化神，也能继续积累长期成长。
- 同境界玩家不再只有数值高低差异，而有玩法倾向差异。

解锁条件：

- 达到元婴初期后，可在 `/me` 或 `/path` 中选择第一条道途。
- 第一条道途免费选择；之后转修有冷却与资源成本。
- 道途不替代境界突破；玩家可以一心突破，也可以边突破边专修。
- 化神突破不强制绑定某条道途，但化神内容会让道途收益更明显。

### 4.2 五类道途

| 道途 | 定位 | 初版加成方向 |
|---|---|---|
| 剑修 | 爆发输出 | 攻击、暴击、世界 Boss 伤害、PvP 先手爆发 |
| 体修 | 生存续航 | 气血、防御、气血回复、秘境连续作战 |
| 丹修 | 补给与突破 | 丹药效果、炼丹产量、突破率、道基不稳恢复 |
| 器修 | 法宝成长 | 炼器词条、强化成本、重铸保底、分解收益 |
| 符阵 | 控制与洞府 | 闭关效率、离线上限、控制抗性、宗门战据点加成 |

道途加成必须小步给：

- 入门只给 2%~3% 量级的方向性收益。
- 小成/大成叠加后形成明显差异，但不能让单一路线成为所有玩法最优。
- PvP 加成要单独审查，必要时在 PvP 中折半或禁用部分 PvE 加成。

内容侧必须给每条道途对应的“瓶颈 sink”，否则二值战斗会把 build 差异抹平成同一条最优数值路线：

| 道途 | 对应内容瓶颈 |
|---|---|
| 剑修 | 世界 Boss chip 伤害、限回合高防目标、贡献榜冲榜 |
| 体修 | 无中途回血的多层秘境、连续遭遇、重伤后恢复效率 |
| 丹修 | 高阶突破率、丹药产量/效果、道基不稳和神魂劫容错 |
| 器修 | 化神法宝词条、强化/重铸成本、法宝分解与 Boss 输出上限 |
| 符阵 | 闭关/道行效率、离线上限、宗门战据点守攻、控制抗性 |

验收标准不是“所有道途都能同样打过同一门槛”，而是每条道途在至少一个长期内容里是合理选择。

### 4.3 道行与道途等级

新增资源：`道行`。

道行来源：

- 元婴期及以上玩家的历练、秘境、世界 Boss、炼制、宗门任务、PvP 周榜都给少量道行。
- 达到版本最高境界且修为已满时，继续闭关所得修为按比例转换为道行。
- 活动副本与飞升试炼给高额但限周的道行。

转换规则初版：

- 化神圆满且修为达到当前阶段成本后，溢出修为的 30% 转为道行。
- 元婴圆满但尚未开放/尚未完成化神突破时，也可把溢出修为的 15% 转为道行，作为对当前封顶玩家的补偿。
- 道行不可交易，不可被夺取。

道途等级：

| 等级 | 定位 | 成本口径 |
|---|---|---|
| 入门 | 选择道途即获得 | 免费 |
| 小成 | 第一阶段差异 | 低额道行 |
| 大成 | 核心 build 成型 | 中额道行 + 道途材料 |
| 圆满 | 长期目标 | 高额道行 + 活动/秘境材料 |
| 宗师 | 版本内软上限 | 周期性飞升点投入 |

### 4.4 转修

转修是“换主职业”，不是清空重练。

规则：

- 玩家可拥有多条道途的历史进度，但同一时间只能激活一条主道途。
- 第一次选择免费；切换主道途需要 `转修令` + 灵石，初版冷却 7 天。
- 切换后保留各道途原有等级，不做经验折损。
- 转修令来源：化神秘境、活动商店、宗门商店；不进入 NPC 直售。
- `转修令` 在 M2 就需要绑定语义。若库存绑定迁移尚未完成，M2 只能把转修令做成不可入袋交易的系统凭证；更推荐在 M2 同步完成最小绑定库存迁移。

这样设计是为了鼓励试 build，而不是惩罚玩家“选错职业”。

## 5. 化神内容

### 5.1 化神历练地图

沿用当前每境界三档地图模型。

| 地图 | 难度 | 精力 | 时长 | 定位 | 独占掉落 |
|---|---|---:|---|---|---|
| 星陨海 | 易 | 16 | 12~15 分钟 | 化神入门稳定刷 | 星陨砂 |
| 幽都裂隙 | 中 | 18 | 15~18 分钟 | 化神中期主刷 | 幽都魂晶 |
| 天外古墟 | 难 | 22 | 18~22 分钟 | 化神后期高风险 | 天外残玉 / 化神丹材料 |

掉落口径：

- 化神当日首买精力基价先定为 `STAMINA_BUY_BASE[4] = 2800`，即 20 精力约 140 灵石/精力。
- 化神地图灵石/精力产出必须显著低于 140，延续反套利测试；目标是比首买成本至少低 25%，避免上线后被反套利红线反复夹住。
- 修为收益明显高于元婴图，避免高阶玩家回头刷低阶成为最优。
- 独占材料主要用于化神丹、化神装备、道途进阶、飞升试炼。

### 5.2 化神秘境：太虚天门

| 配置 | 初版取值 |
|---|---|
| 解锁 | 化神初期 |
| 层数 | 5 |
| 精力 | 60 |
| 入场费 | 1800 灵石 |
| 每日次数 | 2 |
| 主要产出 | 化神丹材料、转修令、道途材料、化神法宝、飞升点材料 |

设计目标：

- 太虚天门定位为高价值日常，但不是挤占全天精力的大头；两次太虚后仍应能兼顾一次 Boss 和少量历练。
- 化神初期能推进 40%~70% 层数。
- 化神圆满稳定通关。
- 掉落以“道途/飞升材料”为主，不只堆灵石。

### 5.3 化神世界 Boss 档

新增 Boss 档 `huashen`。

| Boss | 面向境界 | 持续 | 精力 | 奖励定位 |
|---|---|---:|---:|---|
| 天外魔尊 | 化神 | 2 小时 | 18 | 化神材料、道途材料、飞升点材料 |

沿用当前 Boss 改动：

- 同群独立。
- 按本群已知修仙者人数缩放总血量。
- 参与奖 + 贡献奖 + 小群前列特殊奖励。
- 新增击杀历史记录，但不做跨群总榜。

### 5.4 活动副本

活动副本不阻塞 v2.0，但应在化神上线后尽快补。

初版规则：

- 每周轮换主题，如“天魔潮”“丹霞会”“万剑冢”。
- 活动材料绑定，不进玩家交易。
- 活动商店兑换：称号、转修令、保命符、道途材料、外观型奖励。
- 活动副本给道行，但设周上限，避免肝度失控。

## 6. 飞升点：版本最高境界后的挂机去处

### 6.1 解锁

化神圆满后，玩家可开启“飞升试炼”。

飞升不是三期境界开放，也不把玩家推进炼虚；它是 v2 内的封顶循环，用来承接长期产出。

### 6.2 飞升点获取

来源：

- 化神圆满后的溢出修为转换。
- 每周完成飞升试炼。
- 化神 Boss 前列奖励。
- 活动副本兑换少量飞升材料。

飞升点用途：

- 点亮账号级小幅被动，如气血 +1%、闭关 +1%、炼制加速 +1%。
- 解锁称号、头像文案、群播报特殊文案。
- 推动道途“宗师”级进度。

限制：

- 单项被动设硬上限，初版 5 级。
- 飞升点不可交易。
- 飞升加成进入全局属性计算前，要有总倍率上限，避免旧内容被彻底压扁。

### 6.3 Buff 合算上限

二值战斗下，少量百分比叠加就可能把“0% 门槛”翻成“100% 稳刷”。因此 v2 必须先定合算上限，再调内容。

初版建议：

- PvE 攻击向总倍率上限：同类乘区合算后不超过 `+25%`。
- PvE 生存向总倍率上限：气血/防御/减伤合算后不超过 `+25%`。
- 闭关/道行效率总倍率上限：宗门、道途、飞升、丹药、据点合算后不超过 `+60%`。
- PvP 中所有道途/飞升/据点百分比加成默认折半，并单独维护上限。
- 门槛内容必须按“元婴圆满满 buff 档”和“化神对应阶段满 buff 档”调参，不能按裸锚点调参。

## 7. 经济与玩家交易：仍做，但后置

旧 v2 把坊市/拍卖放在第一优先级。当前应下调，原因：

- 服务器已有玩家触顶，成长目标比交易更紧急。
- 当前灵石买精力反套利空间较窄，玩家交易会放大材料/丹药/活动产出。
- 绑定物、托管物、价格异常审计还没有数据结构基础。

二期仍保留玩家交易，但建议排在化神与道途之后。

### 7.1 交易原则

- 一口价坊市先于拍卖行。
- 材料/丹药先于法宝实例。
- 活动材料、飞升材料、转修令、保命符默认绑定。
- 稀有法宝进入拍卖行前，必须先补交易审计与撤单/结算测试。
- 绑定基建不等于玩家经济；最小绑定库存迁移要随道途/转修提前到 M2，玩家坊市仍可留到 M5。

### 7.2 数据结构注意

当前 `inventory(user_id, item_key, qty)` 不能直接加 `bound` 表示绑定物，因为同一玩家可能同时拥有绑定和非绑定同名物品。

可选方案：

1. 改主键为 `(user_id, item_key, bound)`。
2. 保持主键不变，新增 `bound_qty`。
3. 新增 `inventory_lots`，将交易、绑定、来源都 lot 化。

初版推荐方案 1，迁移简单，服务层改动可控。

## 8. 宗门战与赛季

宗门战和赛季天梯是群活跃增强，不解决“满级无成长”的根因，因此排在化神/道途之后。

### 8.1 宗门战

- 每周固定窗口，初版周六 20:00~21:00。
- 据点 3 个：闭关、掉落、灵石三类加成。
- 成员进攻据点守卫，复用战斗引擎，按宗门累计积分结算。
- 据点 buff 与宗门福利叠加时必须有总上限。

### 8.2 赛季天梯

- 当前周奖池可先保留。
- 二期后段升级为月赛季。
- 赛季末发绑定称号、少量灵石、道行，不发可交易关键材料。

### 8.3 生死斗

生死斗延后到经济审计成熟后再评估。
若做，必须双向确认、限额、冷却、不可夺绑定物，并有日志可查。

## 10. 平衡与验收测试

必须新增/更新的测试：

- 化神配置完整：`REALM_NAMES`、`STAMINA_CAP`、`SECLUSION_STAGE_HOURS`、`BIG_BREAKTHROUGH` 都含 realm 4。
- 化神经济配置完整：`STAMINA_BUY_BASE[4] == 2800`，化神最佳内容灵石/精力低于首买成本至少 25%。
- 元婴圆满可尝试化神突破，缺化神丹时返回 `need_pill`。
- M0 化神地图用“化神初期锚点 + 现役元婴装备/功法”验收；M1 化神装备上线后补化神 GEARED 档并回归重调。
- 化神小怪与 Boss 门槛：元婴圆满满 buff 档可刷化神易图普通内容；打化神中/难图 Boss 仍低于指定阈值，初版 `< 5%`。该满 buff 档必须包含单条拉满道途，例如剑修圆满验攻击门槛、体修圆满验生存门槛。
- 化神门槛按离散口径验收：化神初期易图稳定，化神中/后期分别解锁中/难图稳定刷。
- 化神秘境：化神初期平均推进 40%~70%，化神圆满稳定通关。
- 化神 Boss 击杀挑战次数仍落在目标区间，且小群不会奖励独归榜首。
- 化神圆满后溢出修为可转换道行。
- 道途切换保留历史进度，冷却/费用生效。
- 每条道途至少有一个内容 sink 的收益测试，不用同一张战斗胜率表证明所有 build。
- 飞升点被动有硬上限。
- 满 buff 档仍受 PvE/PvP 合算倍率上限约束。
- 新内容灵石/精力产出仍低于买精力成本，避免“买精力→刷钱”套利。

## 11. 二期里程碑

| 里程碑 | 范围 | 目标 |
|---|---|---|
| **V2-M0 满级止血** | 化神配置、化神丹、化神易/中/难图、元婴→化神突破 | 当前元婴圆满玩家立刻有新目标 |
